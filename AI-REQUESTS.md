# AI-REQUESTS.md

Requests from the AI agent for tools or capabilities that were unavailable during a session.

This file is **not** for package/library dependencies (the AI can install those freely). It is specifically for missing infrastructure -- tools, permissions, or environment capabilities the AI needed but could not use.

## Format

Each entry should include:

- **Date**: When the need arose
- **What was needed**: The tool or capability
- **Context**: What the AI was trying to accomplish
- **Workaround**: How the task was handled without it (if at all)
